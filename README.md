# Anatomy of a Free-Tier Bot Attack

A security field report. How a small SaaS got farmed for free AI compute by a bot, how the real attack door was traced through the payment processor instead of the signup form, and a seven-point checklist for closing the same hole in your own product.

Single static page, no build step. Published with GitHub Pages.

Live page: https://theluckystrike.github.io/free-tier-bot-attack/

Customer identities and the company are anonymized. The dates, command paths, and numbers come from a real production incident.
